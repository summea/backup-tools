NAME

db-backup − a simple program for backing up databases.

SYNOPSIS

db-backup

Before this program is run you will need to make changes to the config/conf file in order to choose what databases you would like backup. If nothing is specified in the conf file... nothing will be backed up.

EXAMPLES

How to Run this Program
If you are using Bash, you should be able to use: ./db-backup

If you aren’t using Bash, you might be able to use: bash db-backup
