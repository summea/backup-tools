NAME

restore − a simple program for restoring files or folders that were backed up with backup.

SYNOPSIS

restore source_file

Before this program is run you will need to make changes to the config/conf file in order to choose what you would like to restore. If nothing is specified in the conf file... nothing will be restored. If you have used the backup program and have not made any changes to the conf file yet, you can safely run the restore program.

EXAMPLES

How to Run this Program
If you are using Bash, you should be able to use: ./restore source_file.tar.gz

If you aren’t using Bash, you might be able to use: bash restore source_file.tar.gz
